# 🚀 Task 1 – Customer Churn Prediction using Machine Learning

## 📌 Overview

This project was developed as **Task 1** of my **AI & Machine Learning Internship at Alfido Tech**.

The objective of this project is to predict whether a customer is likely to leave (churn) a telecom service using supervised machine learning algorithms.

---

## 🎯 Objectives

* Perform data preprocessing and cleaning.
* Analyze customer churn patterns.
* Train multiple machine learning models.
* Compare model performance.
* Evaluate the best-performing model using standard metrics.

---

## 📂 Dataset

* **Dataset:** Telco Customer Churn Dataset
* **Problem Type:** Binary Classification
* **Target Variable:** Churn

---
